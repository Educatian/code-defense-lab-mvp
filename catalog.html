Learning Journey

Code Defense Lab flows from submission to proof of understanding.

This hub maps the product as a learning sequence: students first commit code, then explain it, predict it, adapt it, repair it, and finally reflect on the consistency of their understanding.

Assessment Logic

The main signal is not correctness alone, but whether a student can carry the same logic across explanation, tracing, mutation, and repair.

Start from the student flow to feel the intended learning arc.
